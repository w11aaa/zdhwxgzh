## 三、AI 对话

### 访问
打开 `http://127.0.0.1:7860/chat`

### 支持的问法

| 问法示例 | 系统行为 | 响应速度 |
|----------|----------|----------|
| `北京 事业编` | SQL 关键词精确查询 | <100ms |
| `今日推荐` | 推荐引擎 Top 3 | <100ms |
| `帮助` | 菜单提示 | <10ms |
| `海淀区有没有计算机类的事业编` | **RAG 语义搜索** 公告正文 | 1-3s |
| `有没有不限专业的岗位` | RAG 搜索"专业不限" | 1-3s |
| `快截止的教师岗有哪些` | RAG + 截止日期排序 | 1-3s |
| `国考和省考的区别是什么` | DeepSeek 智能回答 | 3-8s |
| 任意自然语言 | DeepSeek 对话 | 3-8s |

### 对话特性

- **多轮对话上下文**：AI 记住前 10 轮对话，支持追问
- **SSE 流式推送**：实时看到 AI "打字" 过程
- **Tool Badge**：显示 RAG 搜索结果数、KG 推荐数
- **New Chat 按钮**：随时开启新会话

## 四、数据采集

### 控制台操作

在 `http://127.0.0.1:7860` 控制台：

1. 选择数据源（粉笔 / 公考雷达）
2. 选择分类（事业单位 / 公务员 / 国企 / 教师 / 医疗 / 选调）
3. 设置页码和数量
4. 点击「快速采集入库」或「一键补全有效考试」

### 命令行采集

```bash
# 粉笔采集事业单位 50 条
python -m kaoyan_collector.fenbi_crawler --category 事业单位 --max_items 50 --page 1

# 一键补全全国有效公告
python -m kaoyan_collector.fenbi_crawler --backfill_active --max_items 50

# 公考雷达采集
python -m kaoyan_collector.gongkaoleida_crawler --category sydw --max_items 20 --page 1
```

## 五、Agent 功能

### 5.1 Agent Planner（今日草稿）

**控制台操作**: 控制台 → 今日 Agent → 设置数量 → 「一键生成今日草稿」

**命令行**:
```bash
python -m kaoyan_collector.gongkao_today_agent --count 3 --include_attachment_images
```

**工作流**: 推荐 → 检查原公告 → 下载附件 → 生成文章 → 质检 → 提交草稿箱

### 5.2 公众号发布

**控制台操作**: 公告库 → 勾选公告 → 「批量提交草稿」/「预览」

### 5.3 RAG 语义搜索

**命令行测试**:
```bash
python -m kaoyan_collector.rag_engine "北京事业编计算机"
python -m kaoyan_collector.rag_engine --stats    # 查看索引统计
python -m kaoyan_collector.rag_engine --build     # 重建索引
```

### 5.4 知识图谱

```bash
python -c "from kaoyan_collector.kg_engine import KnowledgeGraphEngine; kg=KnowledgeGraphEngine(); kg.build_from_db(); print(kg.stats())"
```

### 5.5 事实一致性评测

```bash
python -m kaoyan_collector.eval_fact_consistency --n 20
python -m kaoyan_collector.eval_fact_consistency --n 10 --json
```

### 5.6 失败诊断

```bash
python -m kaoyan_collector.error_diagnostics --test
python -m kaoyan_collector.error_diagnostics --error "微信返回：{'errcode': 40001}"
```

### 5.7 Agent 测试套件

```bash
python -m kaoyan_collector.test_all_features
```

## 七、API 接口

### 核心 API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/chat/stream?message=xx&session=xx` | GET | SSE 流式对话 |
| `/api/stats` | GET | 数据库统计 |
| `/api/dashboard` | GET | Dashboard 数据 |
| `/api/events?keyword=xx&limit=50` | GET | 公告查询 |
| `/api/recommendations?limit=5` | GET | 推荐公告 |
| `/api/wechat/autoreply/diagnose` | GET | 微信自动回复冲突诊断 |
| `/api/collect` | POST | 触发采集任务 |
| `/api/wechat` | POST | 生成公众号草稿 |

### 示例

```bash
# SSE 流式对话
curl "http://127.0.0.1:7860/api/chat/stream?message=北京事业编&session=abc"

# 公告搜索 API
curl "http://127.0.0.1:7860/api/events?keyword=北京&limit=5"

# 推荐 API
curl "http://127.0.0.1:7860/api/recommendations?limit=3"
```
