## 二、核心类设计

### RAGEngine

```python
class RAGEngine:
    # 单例模式
    _instance = None

    # 属性
    model: SentenceTransformer      # bge-small-zh-v1.5
    index: faiss.IndexFlatIP        # Inner Product 索引
    metadata: List[dict]            # 公告元数据
    _ready: bool                    # 就绪状态

    # 方法
    ensure_ready() -> bool          # 加载/构建索引
    search(query, top_k) -> list    # 混合检索
    _vector_search() -> list        # 向量路
    _keyword_search() -> list        # 关键词路
    _rrf_fusion() -> list           # RRF 融合
    rebuild()                       # 重建索引
```

**关键算法**: RRF 融合

```python
def _rrf_fusion(vec_results, kw_results, k=60):
    score_map = {}
    for rank, r in enumerate(vec_results):
        score_map[r.id] += 1.0 / (k + rank + 1)
    for rank, r in enumerate(kw_results):
        score_map[r.id] += 1.0 / (k + rank + 1)
    return sorted(score_map, by=score, reverse=True)
```

## 七、RAG 索引设计

| 参数 | 值 |
|------|-----|
| Embedding 模型 | BAAI/bge-small-zh-v1.5 |
| 向量维度 | 512 |
| 索引类型 | faiss.IndexFlatIP |
| 距离度量 | Inner Product (归一化后等价余弦) |
| 批量大小 | 32 |
| 索引文件 | `data/rag_index/faiss.index` |
| 元数据文件 | `data/rag_index/metadata.json` |
| 索引大小 | ~1.5MB (737×512×4bytes) |
| 检索耗时 | <50ms |

## 八、安全设计

- **API Key 保护**: `api.md` 不入 Git (.gitignore)
- **数据库**: SQLite WAL 模式，并发读不阻塞
- **审批门**: 高风险操作（发布）必须人工确认
- **质检阻断**: 不合格内容自动阻断，不进入发布流程
- **诊断恢复**: 失败自动诊断 + 建议，减少人工排查
