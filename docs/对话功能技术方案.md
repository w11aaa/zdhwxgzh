## 一、功能概述

为考公信息运营 Agent 系统新增 **Web 智能对话模块**，用户可在 Dashboard 上与 AI 助手自由对话，查询公告、分析数据、触发 Agent 任务。

### 核心能力

| 能力 | 说明 |
|------|------|
| 📋 公告查询 | 自然语言查询数据库（"北京事业编有哪些"） |
| 📊 数据统计 | 实时统计问答（"今天有多少正在报名的"） |
| 🤖 Agent 任务 | 对话触发工具执行（"帮我生成 3 篇草稿"） |
| 🧠 多轮对话 | 基于上下文的连续对话，记住前文 |
| 📝 对话记录 | 保存聊天历史，支持搜索和回放 |

## 三、数据库设计

### 新表: `chat_conversations` (对话会话)

```sql
CREATE TABLE chat_conversations (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    conversation_id TEXT UNIQUE NOT NULL,   -- UUID
    title TEXT NOT NULL DEFAULT '新对话',
    user_id TEXT DEFAULT '',
    model TEXT DEFAULT 'deepseek-v4-flash',
    message_count INTEGER DEFAULT 0,
    created_at TEXT NOT NULL,
    updated_at TEXT NOT NULL
);
```

### 新表: `chat_messages` (对话消息)

```sql
CREATE TABLE chat_messages (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    conversation_id TEXT NOT NULL,          -- 关联 chat_conversations
    role TEXT NOT NULL,                     -- user / assistant / system / tool
    content TEXT NOT NULL,
    tool_calls TEXT,                        -- JSON: 工具调用记录
    token_count INTEGER DEFAULT 0,
    created_at TEXT NOT NULL,
    FOREIGN KEY(conversation_id) REFERENCES chat_conversations(conversation_id)
);
CREATE INDEX idx_chat_conv ON chat_messages(conversation_id, created_at);
```

## 五、核心组件设计

### 5.1 对话引擎 `ChatEngine` (新增文件: `kaoyan_collector/chat_engine.py`)

```python
class ChatEngine:
    """对话引擎：意图识别 → 工具调用 → 回复生成"""

    def __init__(self, db_path, conversation_id):
        self.db_path = db_path
        self.conversation_id = conversation_id
        self.tools = self._register_tools()

    def process_message(self, user_message: str) -> Generator:
        """处理用户消息，流式返回 SSE 事件"""
        # 1. 意图识别
        intent = self._classify_intent(user_message)
        yield {"type": "thinking", "content": f"识别意图: {intent}"}

        # 2. 工具调用
        if intent.needs_tool:
            tool_result = self._execute_tool(intent)
            yield {"type": "tool_result", "result": tool_result}

        # 3. 构建 prompt (注入工具结果)
        prompt = self._build_prompt(user_message, intent, tool_result)

        # 4. 调用 DeepSeek 生成回复
        for chunk in self._stream_llm(prompt):
            yield {"type": "reply_chunk", "content": chunk}

        # 5. 保存消息
        self._save_messages(user_message, full_reply)

        yield {"type": "done", "conversation_id": self.conversation_id}
```

### 5.2 意图分类器

```python
INTENT_PATTERNS = [
    # 模式 → (意图类型, 需要的工具)
    (r"(?:查询|找|搜索|有哪些).*(?:公告|招聘|考试)", "search_events", "search_events"),
    (r"北京|上海|广东|江苏|浙江|...", "search_events_region", "search_events"),
    (r"(?:多少|几个|统计).*(?:报名|公告)", "count_stats", "db_stats"),
    (r"(?:生成|写|做).*(?:草稿|文章|推文)", "generate_draft", "generate_article"),
    (r"(?:今天|今日).*(?:推荐|草稿)", "today_recommend", "recommend_events"),
    (r"帮助|怎么用|功能", "help", None),
]

def _classify_intent(self, message: str) -> Intent:
    """基于正则 + DeepSeek 快速分类的混合策略"""
    # 第一层: 正则快速匹配 (<1ms)
    for pattern, intent_name, tool in INTENT_PATTERNS:
        if re.search(pattern, message):
            return Intent(name=intent_name, tool=tool, confidence=0.9)

    # 第二层: 无歧义通用对话 (不需要工具)
    return Intent(name="general_chat", tool=None, confidence=0.7)
```

### 6.3 核心 JS 逻辑

```javascript
// 发送消息
async function sendMessage() {
    const input = document.getElementById('chat-input');
    const msg = input.value.trim();
    if (!msg) return;

    // 1. 显示用户消息
    appendMessage('user', msg);
    input.value = '';

    // 2. 创建 assistant 消息容器
    const assistantDiv = appendMessage('assistant', '', true);

    // 3. SSE 流式接收
    const eventSource = new EventSource(
        `/api/chat/send?conversation_id=${convId}&message=${encodeURIComponent(msg)}`
    );

    eventSource.onmessage = (event) => {
        const data = JSON.parse(event.data);
        if (data.type === 'reply_chunk') {
            appendToMessage(assistantDiv, data.content);
        } else if (data.type === 'tool_call') {
            appendToMessage(assistantDiv, `\n🔧 正在调用 ${data.tool}...`);
        } else if (data.type === 'done') {
            eventSource.close();
        }
    };
}
```
